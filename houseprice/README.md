# House Price Prediction

A simple pipeline for house price prediction (demo-ready).

**What it does**
- Loads `train.csv` and `test.csv` if present.
- If the CSVs are missing, generates a small synthetic dataset so the script can run for demonstration.
- Performs basic feature engineering, trains a model (XGBoost if available, otherwise RandomForest), evaluates on a validation split, and writes `submission.csv`.
- Produces two plots: predicted price distribution and top feature importances.

**Files**
- [houseprice.py](houseprice.py) : main script
- [train.csv](train.csv) : training data (expected)
- [test.csv](test.csv) : test data (expected)
- [submission.csv](submission.csv) : output predictions (created after run)
